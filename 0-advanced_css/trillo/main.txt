=========================
Main FlexBox Concepts
=========================

Flex container:
The container that its display is set to flex e.g display: flex;
    It can also be set to inline-flex, so that its children acts as inline  elements.

Flex Items:
The children elements in flex container

Main Axis:
The direction which the flex items are laid out.

Cross Axis:
The direction perpendicular to the main axis

==========================
Flex Container Properties
===========================
Flex-direction: row | row-reverse | column | column-reverse
Flex-wrap: nowrap | wrap | wrap-reverse
justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly
